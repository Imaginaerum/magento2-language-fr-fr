Install

Composer is the best way to install the Magento 2 French Pack.

Here are two methods, which facilitate the Magento 2 French Pack installation.

Composer install

Go to your Magento 2 path folder and open a terminal.

composer require Imaginaerum/magento2-language-fr-fr:*
php bin/magento cache:clean

Update

To update:

composer update imaginaerum/magento2-language-fr-fr
php bin/magento cache:clean

Manually install

Download archive from GitHub.

Go to your Magento 2 path.

Create a folder app/i18n/imaginaerum/fr_fr into your Magento 2 path. Unzip it !

Update

Download the latest archive and unzip it into app/i18n/imaginaerum/fr_fr folder.

Configure

You can configure Magento 2 into French for store and admin users.

Frontend configuration

Step 1

Go to Stores > Configuration.

Step 2

Open General > Locale options section.

Choose French (France) into Locale.

Save configuration

Backend configuration for an user

Step 1

Go to System > All Users.

Step 2

Choose the user.

Step 3

Open User Info section.

Choose French (France) into Interface Locale.

Save user
